# 점수 엔진

점수 계산은 `src/engine/scoringEngine.ts`에 구현되어 있습니다.

1. 타자 기본 점수를 계산합니다.
2. 작전 카드 보너스를 계산합니다.
3. 히든젬 보너스를 계산합니다.
4. 캡틴 또는 부캡틴 배수를 적용합니다.
5. 타자 8명의 점수를 합산합니다.
6. 오늘의 마운드 점수를 더합니다.
7. 총점과 선수별 상세 점수를 반환합니다.

타자 기본 점수:

| 기록 | 점수 |
|---|---:|
| 1루타 | 3 |
| 2루타 | 5 |
| 3루타 | 7 |
| 홈런 | 10 |
| 타점 | 2 |
| 득점 | 2 |
| 볼넷 | 2 |
| 도루 | 5 |
| 몸에 맞는 공 | 2 |
| 삼진 | -3 |

오늘의 마운드 점수:

- 5점 차 이상 승리: +50
- 1~4점 차 승리: +30
- 무승부: +10
- 패배: 0
- 무실책: +10
- 실책 1개당: -10
- 상대 득점 2점 이하: +15
- 상대 득점 6점 이상: -10
