# 알려진 제한사항

- 실제 SuperSOL 로그인 또는 네이티브 연동은 없습니다.
- 고객, 계좌, 거래, 개인정보를 사용하지 않습니다.
- 카카오 API를 호출하지 않습니다. 공유는 preview 전용입니다.
- KBO 피드 또는 실제 play-by-play 데이터가 없습니다.
- 실제 선수 이미지나 공식 팀/선수 asset을 사용하지 않습니다.
- 보상 지급 기능이 없습니다.
- 운영 배포 설정은 포함하지 않았습니다.
- AI 감독은 ML 모델이 아니라 deterministic heuristic입니다.
- 미니리그 사용자와 순위는 mock 데이터입니다.
