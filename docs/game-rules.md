# 게임 룰

- 사용자는 KBO 스타일 mock 10개 팀 중 시즌팀 1개를 선택합니다.
- 라인업은 타자 8명입니다: 포수 1명, 센터 내야 2명, 코너 내야 2명, 중견수 1명, 코너 외야 2명.
- 총 예산은 50성입니다.
- 시즌팀 선수는 최소 3명 포함해야 합니다.
- 캡틴은 2배 점수를 받고, 부캡틴은 보너스 배율을 적용하되 최종 점수는 올림한 정수로 표시합니다.
- 캡틴이 결장하면 부캡틴이 2배로 승격됩니다.
- 히든젬은 라인업 안의 3성 이하 선수만 가능합니다.
- 히든젬 보너스는 기본 점수의 30%이며 최대 10점입니다.
- 히든젬은 캡틴 또는 부캡틴과 겹칠 수 없습니다.
- 사용자는 작전 카드 1개와 오늘의 마운드 1팀을 선택합니다.
- 작전 카드 보너스 총합은 최대 20점입니다.
