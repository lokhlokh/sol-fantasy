# 데모 스크립트

1. 앱을 열고 mock 전용 안내를 보여줍니다.
2. 시즌팀 선택 화면에서 KIA 모의 타이거즈를 선택합니다.
3. 홈으로 돌아와 `AI 감독에게 묻기`를 누릅니다.
4. 라인업 화면에서 `AI 자동추천`을 눌러 50성 이하 8명 라인업을 만듭니다.
5. 시즌팀 3명 조건, 예산 바, 역할 규칙을 확인합니다.
6. 작전 선택 화면으로 이동합니다.
7. `장타 노림수` 작전 카드와 원하는 오늘의 마운드를 선택합니다.
8. `오늘의 감독 결정하기`를 누릅니다.
9. 결과 화면에서 내 점수와 AI 감독 점수를 비교합니다.
10. 히든젬 보너스, 작전 카드 효과, 캡틴 배수, 마운드 점수를 설명합니다.
11. 미니리그에서 친구 순위, 배지, 점수 차이를 확인합니다.
12. `카카오 공유 미리보기`를 눌러 HTML 카드 preview를 확인합니다.
13. 카드 화면에서 Lv.1, Lv.2, Lv.3 한솥밥 카드를 보여줍니다.
14. 관리자 화면에서 seed를 바꾸고 같은 seed는 같은 결과를 만든다고 설명합니다.
15. `상태 초기화`로 mock localStorage 상태를 초기화합니다.
